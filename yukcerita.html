PROJECT PORTOFOLIO

YukCerita

YukCerita adalah platform website yang dirancang untuk membantu mahasiswa menangani permasalahan pribadi, akademik, dan sosial. Website ini menyediakan artikel terkait kesehatan mental serta fitur untuk registrasi dan konseling secara online.

Web Development
Membuat website menggunakan Laravel, Bootstrap, dan mengimplementasikan database menggunakan Mysql.
UI/UX Design
Merancang desain (UI) menggunakan Figma, mencakup wireframe dan mockup.
Database
Merancang struktur database menggunakan XAMPP.
